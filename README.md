## Installation

### Prerequisites
- Node.js 18+
- Python 3.10+
- OLLAMA (for local AI)
- Docker (optional)

```bash
# Clone repository
git clone [https://github.com/aman18sharma/SecureSphere](https://github.com/aman18sharma/SecureSphere)
cd SecureSphere
```
## Backend Setup
```bash
cd backend
python -m venv venv

# Linux/MacOS
source venv/bin/activate

# Windows
.\venv\Scripts\activate

pip install -r requirements.txt

```
## Frontend Setup

```bash
cd frontend
